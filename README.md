# LSB Encoder

## 📌 Project Description
The LSB Encoder project is used to hide secret messages inside an image using the Least Significant Bit (LSB) steganography technique. It provides a simple and secure way to encode text into an image without making visible changes.

## 🚀 Features
- Hide a secret message inside an image
- Save the encoded image
- Decode the hidden message
- Simple Python implementation

## 🛠️ Technologies Used
- Python
- Pillow (PIL)

## ▶️ How to Run

1. Install Python
2. Install Pillow

```bash
pip install pillow
```

3. Run the encoder

```bash
python encoder.py
```

4. Run the decoder

```bash
python decoder.py
```

## 📖 What is LSB Steganography?

Least Significant Bit (LSB) steganography is a technique used to hide secret information inside an image by modifying the least significant bits of pixel values. These changes are usually invisible to the human eye.
